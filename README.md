# 🌙 بوابة زاد المؤمن (Zad Al-Momen)

بوابة روحية وتفاعلية شاملة تهدف إلى تقريب العلوم الشرعية، الأذكار اليومية، وفهم خريطة الدعاء وآدابه بأسلوب عصري وسلس، مدعومة بمساعد ذكاء اصطناعي شرعي موثق.

🔗 **رابط المعاينة المباشرة:** [Zad Al-Momen Live](https://omar-i3.github.io/Zad-Al-Momen/)

---

## ✨ أبرز مميزات المشروع

* 🤲 **خريطة الدعاء التفاعلية (`duaa.html`):** رحلة استكشافية شاملة لتعريف الدعاء، علاقته بالقدر المبرم والمعلّق، أسباب الاستجابة وموانعها، وشبهات معاصرة.
* 📿 **ركن الأذكار اليومية (`azkar.html`):** أذكار الصباح والمساء واليوم والليلة مقسمة في أكورديون تفاعلي مزود بـ **عداد إلكتروني محلي** لكل ذكر.
* 🤖 **مساعد تبصرة الرقمي (AI Assistant):** شات بوت ذكي متصل بـ **DeepSeek API** يجيب على الأسئلة الشرعية والفقهية مدعماً بالأدلة من القرآن والسنة والتخريج المباشر.
* 💾 **ذاكرة محادثة مستمرة (Persistent Chat):** حفظ سياق الشات والمحادثات عبر `localStorage` لضمان عدم فقدان السوالف عند التنقل بين صفحات الموقع.
* 🔍 **تدقيق حديثي مباشر:** محرك بحث سريع مرتبط بموسوعة **الدرر السنية** للتحقق من صحة الأحاديث.
* 🎨 **واجهة مستخدم زجاجية (Glassmorphism):** تصميم يتضمن خلفية نجوم تفاعلية (`Canvas`) ودعم كامل للوضعين الليلي والنهاري (Light/Dark Mode).

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **Front-End:** HTML5, CSS3 (Glassmorphism & Responsive Design), Vanilla JavaScript (ES6+).
* **AI Model:** DeepSeek-Chat API.
* **Serverless Backend:** Cloudflare Workers (CORS & API Key Encryption).
* **Hosting:** GitHub Pages.
